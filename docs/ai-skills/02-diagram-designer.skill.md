# Skill 02 — Diagram Designer

## Purpose

Create simple, elegant diagrams for the workshop.

## Diagram types to prefer

1. Linear flow: for processes.
2. Two-column comparison: for myths vs reality.
3. Layered architecture: for internal systems.
4. Formula diagram: for concepts like agent = model + tools + rules.
5. Semáforo: for risk classification.
6. Staircase: for maturity roadmap.

## Diagram rules

- 5 to 7 nodes maximum.
- No crossing lines.
- Use short labels.
- Avoid dense architecture charts.
- Include a plain-language caption.
- Produce Mermaid first.
- Produce Excalidraw layout guidance after Mermaid.

## Output format

```markdown

Purpose:

...
