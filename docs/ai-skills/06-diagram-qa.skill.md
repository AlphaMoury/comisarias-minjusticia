# Skill 06 — Diagram QA

## Purpose

Validate diagrams before they are used in slides.

## Review dimensions

1. Conceptual accuracy.
2. Simplicity.
3. Visual hierarchy.
4. Privacy/safety message.
5. Relevance to Comisarías.
6. Exportability to Mermaid/Excalidraw.

## QA protocol

For each diagram, answer:

- What does this diagram teach?
- Could the audience misunderstand it?
- Is any arrow implying illegal or unsafe data access?
- Does it preserve human review?
- Can it be redrawn in 30 seconds on a whiteboard?

Reject diagrams that fail any of these criteria.
