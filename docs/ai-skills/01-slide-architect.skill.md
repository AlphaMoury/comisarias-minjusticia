# Skill 01 — Slide Architect

## Purpose

Transform complex AI/legal/public-sector ideas into simple, slide-ready explanations for non-technical audiences.

## Rules

- One idea per slide.
- Maximum 3 visible bullets.
- Prefer one diagram over five bullets.
- Use Spanish.
- Use institutional but plain language.
- Add speaker notes with the deeper explanation.
- Include estimated timing.

## Slide schema

For every slide, produce:

```yaml
slide_number:
title:
main_message:
visible_text:
diagram:
speaker_notes:
time_minutes:
interaction:
```

## Quality checklist

- Can a non-technical official understand it in 10 seconds?
- Does it avoid jargon?
- Does it connect to daily work in a Comisaría?
- Does it preserve the message of human review?
