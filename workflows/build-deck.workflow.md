# Workflow — Build Deck

## Goal

Create a complete 25-slide deck for a 2-hour workshop on AI for Comisarías de Familia.

## Phase 1 — Understand

Read:

- `AGENTS.md`
- `docs/ai-context/brief_codex_ia_comisarias.md`
- `docs/ai-skills/*.md`
- `examples/key-diagrams.mmd`

Produce:

- One-paragraph understanding.
- List of expected artifacts.
- Risk assumptions.

## Phase 2 — Outline

Create a slide outline with:

- Slide number.
- Title.
- Main message.
- Diagram need.
- Time estimate.

Target total: 120 minutes.

## Phase 3 — Diagrams

For each key diagram:

- Generate Mermaid.
- Generate Excalidraw layout guidance.
- Generate speaker note.
- Run Diagram QA.

## Phase 4 — Demos

Generate 4 safe demos:

1. Anonymization.
2. Transcript to act.
3. Deadline traffic light.
4. Safe legal prompt.

Each demo must include fictitious inputs and expected output.

## Phase 5 — Build

Build the final deck in the chosen format.

Acceptable outputs:

- `deck.md`
- `deck.mdx`
- `deck.html`
- `deck.pptx`
- `diagrams/*.mmd`
- `diagrams/*.svg`

## Phase 6 — Review

Run the accessibility and safety checklist.

Produce:

- Final deck.
- Speaker notes.
- Diagram index.
- Demo prompts.
