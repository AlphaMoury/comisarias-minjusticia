# Workflow — Refine Diagrams

## Goal

Improve diagrams until they are clear enough for a non-technical audience.

## Steps

1. List all diagrams.
2. For each diagram, explain the message in one sentence.
3. Remove unnecessary nodes.
4. Convert dense diagrams into simpler flows.
5. Produce Mermaid and Excalidraw layout instructions.
6. Run Diagram QA.
7. Export or prepare for export.

## Acceptance criteria

- Each diagram has one message.
- Each diagram can be understood in 10 seconds.
- No diagram implies unsafe access to sensitive systems.
- No diagram suggests that AI replaces the official decision-maker.
