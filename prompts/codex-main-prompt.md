# Prompt for Codex

Read `AGENTS.md`, `docs/ai-context/brief_codex_ia_comisarias.md`, all skill files in `docs/ai-skills/`, and `workflows/build-deck.workflow.md`.

Your task is to build a complete slide deck for the workshop “IA para Comisarías de Familia: del uso informal al copiloto institucional seguro”.

First, produce a concise implementation plan and list the files you will create.

Then generate:

1. A 25-slide deck in Markdown.
2. Speaker notes.
3. A timing table totaling 120 minutes.
4. Mermaid diagrams for each key concept.
5. Excalidraw layout guidance for each diagram.
6. Four live demo scripts using fictitious data.
7. A final QA checklist.

Important:

- Do not use real data.
- Do not invent legal citations.
- Mark legal references as “verificar” unless provided.
- Keep slides visual and simple.
- Use Spanish.
